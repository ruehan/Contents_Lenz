# Contents Lenz - AI 기반 콘텐츠 요약 및 분석 도구

Contents Lenz는 GPT-4o Mini 모델을 활용한 콘텐츠 요약 및 분석 도구입니다. 긴 문서, 기사, 웹 페이지 등을 쉽게 요약하고 분석할 수 있습니다.

## 개발 현황 (테스트 버전)

현재 Contents Lenz는 테스트 단계에 있으며, 다음과 같은 기능이 구현되어 있습니다:

### 구현 완료된 기능

- **콘텐츠 입력 및 로드**

  - 텍스트 직접 입력
  - 파일 업로드 (TXT, PDF, DOCX)
  - URL 입력을 통한 웹 콘텐츠 스크래핑

- **요약 기능**

  - 요약 길이 조절 (짧은 요약, 중간 길이, 상세 요약)
  - 주요 키워드 추출 및 표시
  - 다국어 지원: 원본 언어 감지 및 선택한 언어로 요약 출력

- **UI/UX**

  - 일렉트론 기반 데스크톱 애플리케이션
  - 반응형 디자인 (다양한 화면 크기 지원)
  - 보라색 테마 적용
  - 탭 기반 인터페이스 (텍스트 입력, 파일 업로드, URL 입력)
  - 요약 결과 및 키워드 표시 영역 개선
  - 로딩 인디케이터 및 상태 표시

- **API 서버**
  - FastAPI 기반 RESTful API
  - 텍스트 요약 엔드포인트
  - 웹 콘텐츠 스크래핑 엔드포인트
  - 키워드 추출 엔드포인트
  - 언어 감지 기능

### 개발 예정 기능

- **고급 분석 기능**

  - 감정 분석 (긍정/부정/중립)
  - 주요 주제 자동 태깅 및 분류
  - 텍스트 가독성 평가
  - SEO 최적화 제안

- **추가 콘텐츠 소스**

  - 유튜브 영상 URL을 통한 자동 트랜스크립트 분석
  - 소셜 미디어 포스트 분석

- **결과 출력 및 공유 확장**

  - 결과 저장 형식 확장 (PDF, DOCX, TXT, HTML)
  - 이메일 공유 옵션
  - 소셜 미디어 공유 통합

- **UI/UX 개선**
  - 다크 모드 지원
  - 사용자 설정 저장 기능
  - 히스토리 기능 (이전 요약 결과 저장 및 불러오기)
  - 더 다양한 언어 지원

## 언어 지원

Contents Lenz는 다양한 언어로 된 콘텐츠를 처리할 수 있습니다:

- **자동 언어 감지**: 입력된 텍스트의 언어를 자동으로 감지합니다.
- **원본 언어 유지**: 기본적으로 원본 텍스트와 동일한 언어로 요약합니다.
- **출력 언어 선택**: 원하는 언어로 요약 결과를 받을 수 있습니다 (한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어, 러시아어, 포르투갈어, 이탈리아어, 네덜란드어, 아랍어, 힌디어, 베트남어, 태국어).

이 기능을 통해 외국어로 된 문서를 빠르게 이해하거나, 다국어 환경에서 콘텐츠를 공유할 때 유용하게 활용할 수 있습니다.

## 설치 방법

### 방법 1: 실행 파일 다운로드 (권장)

가장 간단한 방법은 GitHub 릴리스 페이지에서 운영체제에 맞는 실행 파일을 다운로드하는 것입니다:

- **Windows**: `Contents-Lenz-Setup-x.x.x.exe` 파일을 다운로드하여 실행합니다.
- **macOS**: `Contents-Lenz-x.x.x.dmg` 파일을 다운로드하여 설치합니다.

[GitHub 릴리스 페이지](https://github.com/ruehan/Contents_Lenz/releases)에서 최신 버전을 다운로드할 수 있습니다.

### 방법 2: 소스 코드에서 설치

개발자나 소스 코드를 직접 수정하고 싶은 사용자를 위한 방법입니다:

1. 저장소 클론

```bash
git clone https://github.com/ruehan/Contents_Lenz.git
cd Contents_Lenz
```

2. 필요한 패키지 설치

```bash
# API 서버용 패키지 설치
pip install -r requirements.txt

# 일렉트론 UI용 패키지 설치
cd electron
npm install
```

3. OpenAI API 키 설정

`.env` 파일을 프로젝트 루트 디렉토리에 생성하고 다음과 같이 API 키를 설정합니다:

```
OPENAI_API_KEY=your_api_key_here
```

## 사용 방법

### 방법 1: 실행 파일로 사용 (권장)

릴리스 페이지에서 다운로드한 실행 파일을 사용하는 경우:

1. 설치된 애플리케이션을 실행합니다.
2. 텍스트 입력, 파일 업로드 또는 URL 입력을 통해 콘텐츠를 요약할 수 있습니다.

> 참고: 실행 파일 버전에서는 OpenAI API 키를 처음 실행 시 입력하거나 설정 메뉴에서 변경할 수 있습니다.

### 방법 2: 소스 코드에서 실행

소스 코드에서 직접 실행하는 경우:

#### 일렉트론 UI 실행 (권장)

일렉트론 기반 데스크톱 UI를 사용하는 것이 가장 편리합니다:

```bash
# API 서버 실행
python run_api_server.py

# 새 터미널에서 일렉트론 앱 실행
cd electron
npm start
```

#### API 서버만 실행

API 서버를 실행하여 다양한 클라이언트에서 사용할 수 있습니다:

```bash
python run_api_server.py --host 0.0.0.0 --port 8000 --reload
```

API 서버가 실행되면 다음 URL에서 API 문서를 확인할 수 있습니다:

- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

## 테스트 결과 및 피드백

현재 테스트 버전에서는 다음과 같은 사항을 확인했습니다:

- **성능**: GPT-4o Mini 모델을 사용하여 빠르고 정확한 요약 결과를 제공합니다.
- **다국어 지원**: 다양한 언어의 텍스트를 성공적으로 요약하고 번역합니다.
- **웹 스크래핑**: 대부분의 웹사이트에서 콘텐츠를 성공적으로 추출하고 요약합니다.
- **UI/UX**: 직관적인 인터페이스로 사용자가 쉽게 이용할 수 있습니다.

### 개선 필요 사항

- 일부 복잡한 레이아웃의 웹사이트에서 콘텐츠 추출 정확도 향상 필요
- 대용량 파일 처리 시 성능 최적화
- 오프라인 모드 지원 검토
- 더 다양한 파일 형식 지원 (예: EPUB, HTML)

## 개발 로드맵

- **1단계 (완료)**: 기본 버전 - 핵심 기능 개발 및 GPT-4o Mini API 연동, 텍스트 입력 및 기본 요약 기능 구현
